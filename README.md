

# CE-ESY-ASSIGNMENTS-
هذا المشروع يقدّم تنفيذًا لمخزن دائري (Circular Buffer) بلغة C، ويعرض كيفية التعامل مع عمليات التهيئة، الكتابة، القراءة، وحالات الامتلاء والفراغ داخل المخزن. كما يطبّق البرنامج المطلوب في الدالة الرئيسية باستخدام اسم المستخدم. ما هو المخزن الدائري؟

المخزن الدائري هو مصفوفة ثابتة الحجم، لكن طريقة التعامل معها تجعل المؤشرات تلتف عند الوصول لنهاية المصفوفة.  
يتم التحكم بالمخزن باستخدام ثلاث قيم:

- head: موقع القراءة  
- tail: موقع الكتابة  
- count: عدد العناصر داخل المخزن  

عند وصول المؤشر لنهاية المصفوفة، يتم استخدام العملية:

(index + 1) % SIZE
لإعادته إلى البداية.

## 🛠️ شرح الدوال الأساسية

### 1) init() – تهيئة المخزن  
تعيد المؤشرات إلى الصفر وتضع المخزن في حالة فارغة.

### 2) isFull() – التحقق من الامتلاء  
يرجع True إذا كان:
count == SIZE
### 3) isEmpty() – التحقق من الفراغ  
يرجع True إذا كان:
count == 0
### 4) writeBuffer() – الكتابة داخل المخزن  
- إذا كان المخزن ممتلئًا → تظهر رسالة Overflow  
- يتم تخزين الحرف في موقع tail  
- يتم تحريك tail بشكل دائري  
- يتم زيادة count  

### 5) readBuffer() – القراءة من المخزن  
- إذا كان المخزن فارغًا → تظهر رسالة Underflow  
- يتم قراءة الحرف من head  
- يتم تحريك head بشكل دائري  
- يتم تقليل count  

الدالة الرئيسية تنفّذ المطلوب في الواجب:

1. تهيئة المخزن الدائري  
2. قراءة اسم المستخدم من الإدخال  
3. إضافة النص "CE-ESY" إلى الاسم  
4. كتابة كل حرف داخل المخزن  
5. قراءة الحروف من المخزن وطباعة الناتج  
6. التأكد أن المخزن أصبح فارغًا بعد القراءة  
7. تجربة حجم مخزن صغير وكبير لاختبار حالتي Overflow وUnderflow
